`startTransition` lets you update the state without blocking the UI.

```jsx
startTransition(scope)
```

[官方文档链接](https://react.dev/reference/react/startTransition)

`startTransition` 属于 `useTransition` 的一部分，唯一区别在于 `startTransition` 并非 `hook` 因此可以在 `function component` 之外使用.
