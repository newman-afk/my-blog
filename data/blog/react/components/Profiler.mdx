`<Profiler>` lets you measure rendering performance of a React tree programmatically.

```jsx
<Profiler id="App" onRender={onRender}>
  <App />
</Profiler>
```

[官方文档链接](https://react.dev/reference/react/Profiler)

请直接使用 React Developer Tools.
